Machine Learning project for customer churn prediction using classification models and sample dataset.
# Streamify Customer Churn Prediction

## Objective
Predict whether a customer will churn using machine learning classification models.

## Models Used
- Logistic Regression
- Random Forest Classifier

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Business Goal
Help Streamify identify customers at risk of cancelling their subscriptions.
